WebGL experiments

This is a collection of my attempts to learn WebGL. I don't guarantee that all of it is good code - some of the tutorials I followed contained some pretty bad code. But rather than refactor in place, I tended to create a new experiment and improve the code there.

Experiment 1
Mostly following along with IndigoCS WebGL Tutorial 1 . It's a simple colored triangle, but a complete WebGL program. I restructured his code a little, but for the most part, it's a directly copy.

Experiment 2
First half of IndigoCS WebGL Tutorial 2. Animation and rotation of the colored triangle from Experiment 1

Experiment 3
Second half of IndigoCS WebGL Tutorial 2. Move from a simple triangle to a full cube. Includes backface culling and index buffers.

Experiment 4
Implementation of IndigoCS WebGL Tutorial 3. Covers texture mapping on the cube from Experiment 3

Experiment 5
Implementation of IndigoCS WebGL Tutorial 4. Loading external geometry and textures. Not really happy with code organization at this point; I broke the code up a bit into separate functions, but it really needs to be separate modules.
